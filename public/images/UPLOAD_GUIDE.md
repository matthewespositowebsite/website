## üöÄ How to Upload

1. **Rename your photos** with the exact filenames above
2. **Copy all images** to `public/images/` folder
3. **Commit and push**:
   ```bash
   git add public/images/
   git commit -m "Add portfolio images"
   git push
   ```
4. **Wait 2-3 minutes** for automatic deployment

## üí° Tips

- **File format**: JPG (JPEG)
- **Quality**: 85-90%
- **Optimize**: Use TinyPNG or Squoosh to reduce file size
- **Max size**: ~500KB per image
- **All lowercase**: Use exact names (case-sensitive)
